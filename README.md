# Getting Started

1. Clone the repository:
```bash
  git clone https://github.com/Abdrakhmanov-Renat/2048_game.git
  cd 2048_game
```
2. Install dependencies:
```bash
npm install
# or
yarn install
```
3. Run the project locally:
```bash
npm start
# or
yarn start
```
